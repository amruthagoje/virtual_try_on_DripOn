# DripON

This is a Next.js application that enables users to virtually wear different clothes using AR lenses, capture their experience, and share it using gestures.

Built with Firebase Studio.

## Google Technologies Used

This project is built and powered by a suite of modern Google technologies:

*   **Firebase App Hosting:** Provides a fully-managed, secure hosting environment for the Next.js application, with features like automatic deployments from a Git repository.
*   **Firebase Authentication:** Handles user identity with easy-to-use and secure authentication services, including anonymous sign-in for a seamless initial user experience.
*   **Firestore:** A flexible, scalable NoSQL database for storing and syncing application data, such as captured media, in real-time.
*   **Google Gemini:** The core intelligence behind the gesture-based media transfer feature, interpreting user actions to initiate sharing.
*   **Genkit:** This project uses Genkit, an open-source framework from Google, to build and manage the AI flow for gesture recognition, ensuring a robust and scalable implementation.
