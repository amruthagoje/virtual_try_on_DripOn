# 🚀 DripOn - A Virtual TryON

## 📌 Overview

DripOn is an innovative Virtual Try-On system that allows users to try clothes digitally without physically wearing them. Using camera input and intelligent processing, users can visualize how different outfits look on them in real time.
This project aims to revolutionize online shopping by reducing uncertainty in clothing selection and enhancing user experience.

---

## 🎯 Features

* 👕 Virtual Try-On – Try clothes in real-time using camera
* 🎥 Live Preview – Instant outfit visualization
* 📸 Capture Photos & Videos – Save your looks
* 🤳 Gesture-Based Interaction (optional)
* 📤 Media Sharing – Transfer images/videos
* 🧠 Smart Fitting System – Adjust clothes to body shape

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript, React
* **Backend:** Python, Node.js
* **Database:** Firebase, MongoDB
* **Tools & Platforms:** Git, GitHub, VS Code

---

## ⚙️ How It Works
1. 📷 User opens the app and enables camera
2. 🧍 Body is detected using computer vision
3. 👚 Selected clothes are overlaid on the body
4. 🎯 System adjusts fitting in real time
5. 📸 User captures or records output

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/amruthagoje/DripOn.git
```

### 2️⃣ Navigate to project folder

```bash
cd DripOn
```

### 3️⃣ Install dependencies

```bash
npm install
```

or

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the project

```bash
npm start
```

or

```bash
python app.py
```
