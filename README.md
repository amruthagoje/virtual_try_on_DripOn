# DripON

This is a Next.js application that enables users to virtually wear different clothes using AR lenses, capture their experience, and share it using gestures.

Built with Firebase Studio.

## Google AI Tools Integrated

This project leverages the power of Google's AI to provide innovative features:

*   **Google Gemini:** The core intelligence behind the gesture-based media transfer feature, interpreting user actions to initiate sharing.
*   **Genkit:** This project uses Genkit, an open-source framework from Google, to build and manage the AI flow for gesture recognition, ensuring a robust and scalable implementation.
